# Analysis Notebooks

This directory contains Jupyter notebooks for data exploration, analysis, and modeling.

## Notebooks

- **task1_data_exploration.ipynb**: Data exploration, schema interpretation, and enrichment pipeline
- **task2_eda.ipynb**: Exploratory data analysis with trend analysis and policy insights

## Usage

1. Ensure you have activated the virtual environment
2. Start Jupyter: `jupyter notebook`
3. Open the desired notebook
4. Run cells sequentially

## Outputs

- Enriched datasets are saved to `data/processed/`
- Figures are saved to `reports/figures/`
- Insights are documented in `reports/insights.md`
